Directory Structure
===================

* db2vim     - docbook to vimhelpfile converter
* htdocs     - some parts of the vim-latex website
* TODO       - TODO files (currently not really used)
* vimfiles   - the actual code of vim-latex, you can use this as your ~/.vim
               folder ;-)
* vim-latex  - previous location of some documentation, will be merged into
               vimfiles/doc
